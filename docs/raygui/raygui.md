Basic Components

Label
Displays static text. Used for titles, captions, and read-only information.

Button
Clickable button that triggers an action when pressed.

LabelButton
Text that behaves like a button. Visually lighter than a normal button.

Toggle
A button that maintains an on/off state.

ToggleGroup
A collection of toggle buttons where one item is typically selected at a time.

ToggleSlider
A toggle control displayed as a slider-like selector between options.

CheckBox
A checkbox used to enable or disable a feature.

ComboBox
Compact selector that allows choosing one item from a list.

DropdownBox
Dropdown menu that expands to show available choices.

TextBox
Single-line text input field.

TextBoxMulti
Multi-line text input area for larger text content.

ValueBox
Text field specialized for editing numeric values.

Spinner
Numeric editor with increment and decrement buttons.

Slider
Adjusts a numeric value by dragging a handle along a track, usually with labels.

SliderBar
Simplified slider without extra labels.

SliderPro
Advanced configurable slider with more control over appearance and behavior.

ProgressBar
Displays progress toward a value or completion percentage.

StatusBar
Displays status information, messages, or application state.

Grid
Draws a coordinate grid and can be used for snapping and editor layouts.

DummyRec
Invisible placeholder control useful for spacing, layout, and debugging.

Containers

WindowBox
Window container with a title bar and close button.

GroupBox
Visual grouping container used to organize related controls.

Panel
Simple background container used to separate UI sections.

ScrollPanel
Scrollable container that allows viewing content larger than its visible area.

TabBar
Displays tabs for switching between different views or pages.

Line
Horizontal separator line used to visually divide sections.

Advanced Controls

ListView
Scrollable list of text items with selection support.

ListViewEx
Extended list view that works directly with arrays and dynamic data.

ColorPicker
Complete color selection widget including hue, saturation, value, and alpha controls.

ColorPanel
Main color selection area used inside color pickers.

ColorBarHue
Hue selection bar used to choose a color family.

ColorBarAlpha
Transparency selection bar used to control alpha values.

MessageBox
Modal dialog used to display messages and receive user confirmation.

TextInputBox
Modal dialog containing a text input field and action buttons.
